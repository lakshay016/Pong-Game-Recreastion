A fully functional recreation of the classic **Pong** game, built in **Python** using the **Pygame** library.

---

## ‚ú® Features

- **Classic Pong gameplay** with smooth paddle and ball movement  
- **Three difficulty modes** (Easy, Medium, Hard) with dynamic ball and paddle speeds  
- **Collision physics** for paddle hits, wall bounces, and scoring  
- **Start menu & UI screens**, including:
  - Main menu  
  - How-to-play instructions  
  - Difficulty selection  
  - Score display  
  - Win/loss screen with restart option

- **AI-assisted paddle movement** for higher difficulty levels  
- **Clean modular code** using functions for game reset, movement, and event handling
